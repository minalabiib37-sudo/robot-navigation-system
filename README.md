# Autonomous Robot Navigation System

## Overview

This project was developed as my senior engineering project during my Bachelor's degree in Electrical and Communication Engineering.

The objective was to design and implement an autonomous mobile robot capable of navigating unknown environments using computer vision, embedded systems, and intelligent path planning algorithms.

---

## Features

- Autonomous navigation
- Computer vision-based obstacle detection
- Path planning
- Embedded system integration
- Real-time sensor processing
- AI-assisted decision making
- A* path planning algorithm
- Grid-based simulation
- Text-based path visualization
- Start and goal position tracking
